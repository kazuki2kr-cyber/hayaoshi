# Sparks Station

海外のMicro-SaaS開発・売却事例から「稼げるヒント」をエンジニアへ届ける分析メディア & プロダクトスタジオ。

## Overview
- **Portal**: Insights for developers (`src/app/(portal)`)
- **Products**: In-house Micro-SaaS products (e.g., Fantasy Quizzes Kingdom) (`src/app/quiz`)

## Stack
- Next.js (App Router)
- Tailwind CSS (v4)
- React Markdown (remark-gfm)
- Firebase (Auth/Firestore)

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
